* **Smart Itinerary Generator (Demo):** An interactive JavaScript-based tool that generates detailed day-by-day travel plans based on location (Bali, Malang, etc.), duration, and budget.

* **Dynamic App Showcase:** Automatic slideshow carousel demonstrating the mobile app interface flow.
* **Interactive Components:** Sticky navigation, smooth scrolling, mobile drawer menu, and accordion-style feature toggles.
* **Professional Assets:** Curated imagery and custom brand assets organized in a clean directory structure.
